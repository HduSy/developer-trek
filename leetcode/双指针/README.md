# 双指针题解
在处理数组和链表相关问题时，双指针技巧是经常用到的（对数组而言指针看作是索引），技巧分为两类：左右指针和快慢指针
## 左右指针
- 相向移动
- 相背移动
## 快慢指针
- fast 快指针，移动得快
- slow 慢指针，移动得慢
